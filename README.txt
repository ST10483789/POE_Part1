## Project Overview
This project is a Cybersecurity Awareness Chatbot developed using C# Windows Forms.

The chatbot is designed to educate users about common cybersecurity topics and provide basic cybersecurity awareness through an interactive graphical user interface (GUI).

## Features
- Graphical User Interface (GUI)
- Cybersecurity-themed chatbot
- Keyword recognition
- Randomized responses
- Sentiment detection
- User memory functionality
- Error handling and input validation
- Timestamped chat messages
- Enter key support for sending messages
- Multiple cybersecurity topics

## Cybersecurity Topics Covered
- Password Security
- Phishing
- Malware
- Privacy
- VPN Security
- Online Scams

## Technologies Used
- C#
- Windows Forms
- Visual Studio

## How to Run the Application
1. Open the project in Visual Studio.
2. Build the solution.
3. Run the application using Start or F5.
